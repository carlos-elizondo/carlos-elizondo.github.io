Created

Description

GitHub Pages only host static pages, this is why for the implementation of the Scrabble game you see, it only contains the Scrabble interface. It does not contain the backend code that handles the game logic. This being the game score and the 2 player modality.

This implementation does contain persistence which you will notice once you place a word on the board and then refresh the page. The word you place will still be there. This takes advantage of the browser's local storage.

The board and the rack are programmatically created using JavaScript.

The backend implementation uses Node.js and Express.js.

Instructions

  • Every player will start their turn by drawing seven tiles from the Scrabble bag.
  • Blanks: The two blank (*) tiles may be used as any letters. When playing a blank, you must state which letter it represents. It remains that letter for the rest of the game.
  • When looking at the board, players will see that some squares offer multipliers. Should a tile be placed on these squares, the value of the tile will be multiplied by 2x (cyan) or 3x (blue). Some squares will also multiply the total value of the word by 2x (pink) or 3x (red) and not just the single point value of one tile.
  • You can place a word horizontally or vertically.
  • Use the x and y coordinates to choose where to place the word coordinates start from 1 to 15. .
  • Use the drop-down menu to choose placement orientation.
  • Use the play button to place the word.
  • Use the reset button to start a new game.
  • Use the help button to get a word suggestion from your current tiles.
