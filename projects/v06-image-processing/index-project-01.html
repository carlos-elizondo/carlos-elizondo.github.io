From Pixels to Mosaics: A Comprehensive Exploration of Image Processing Techniques

In this project I implemented several image processing algorithms, including pixel operations, convolutions, dithering and mosaicing.

Description

The code uses the open source p5.js library for loading, saving and displaying images.

The following pixel operations are supported: brightness, contrast, and saturation.

The following convolutions are supported: Box Blur, Gaussian Blur, Sharpen, and Edge Detection.

The following dithering options are supported: uniform, random, and ordered.

Instructions

  • You can use the control panel to your right to perform pixel operations, image convolutions, dithering, and image mosaics.
  • You can choose from several images using the drop-down menu at the top of the control panel.
  • There is also a drop-down menu for mosaic which lets you choose from “movie” or “album” covers and the size of each mosaic.
  • There is a reset button just before the image convolution section.
