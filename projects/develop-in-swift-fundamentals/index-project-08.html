Applications Developed (13 apps)

1. IBBasics

A simple iOS app that teaches a user how to use the Interface Builder on Xcode and the basics of outlets and actions while using a storyboard. When the button is pressed its tint color changes from blue to red.

2. Light

On this app, when a user taps the screen, the background color changes from white to black and vice versa. Outlets and actions.

3. Hello

An About me app - In this app, I learn how to configure views using Interface Builder, how to customize a label, and how to customize an image view.

4. Hobby Tutorial

A hobby screen using layout guides to set up a clean, well-balanced interface.

5. Set & Clear

An app that that goes over the various components of UIKit such as buttons, switches, sliders, text fields, and labels using the Interface Builder.

6. Calculator Layout

Learning about Auto Layout and Stack Views to make sure apps look good on every iOS device by building a calculator layout app.

7. Apple Pie

Building a simple word-guessing game, like Hangman. Each player has a limited number of turns to guess the letters in a word. Each incorrect guess results in an apple falling off the tree. The player wins by guessing the word correctly before all the apples are gone.

8. Traffic Segues

Learning about segues and navigation controllers. And how to use segues to transition from one view controller to another.

9. Login

Creating a login screen that passes a username between view controllers. I use view controllers, a navigation controller, and segues to create both the login screen and a simple landing screen that displays in its title either the username or text related to a forgotten username or password.

10. Rainbow Tabs

Learning about tab bar controllers to organize different kinds of information or different modes of operation. I learn how to appropriately use a tab bar controller and how to customize tab bar items.

11. About Me

Using the tab bar controller to display distinct types of information about myself in separate tabs. This app is similar to the "Hello" app. I am using auto layout and stack views to make sure the app looks good on all iOS devices.

12. Order Of Events

Visualizing the view's life cycle. This app adds/appends to the text of a label based on the events in the view controller life cycle.

13 . Personality Quiz

This app is a personality quiz - players are presented with a topic and answer questions that align them to a particular outcome. Here I am using segues and navigation controllers to transition between view controllers and pass data between them. And I use various UIKit components.
