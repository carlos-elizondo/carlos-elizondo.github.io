Applications Developed (8 apps)

1. Cat Photo Frame

A simple iOS app that displays a single photo. The user interface is created with the Interface Builder on a storyboard.

2. Question Bot

On this app, I was tasked to develop the "brain" of the chatbot. The chatbot has a quirky personality answering your question with another question.

3. Bouncy Ball

For this app, I used SpriteKit and GameplayKit to create the shapes and the physics of the game. The objective is for the ball to touch every single orange square in the screen to win, by bouncing them around four red platforms.

4. Meme Maker

Simple meme maker app. The user can add text to the top and bottom of the photo with a segmented control. Using a pan gesture recognizer, the user can move the text around the screen.

5. Rock, Paper, Scissors

Model-View-Controller (MVC) design pattern in Swift. Using Xcode's Interface Builder. The components change depending on the state of the game.

6. Color Mix

An app that uses switches and sliders to mix colors that must be connected with actions and outlets to the views and controls.

7. ChatBot

This is a reworked version of the Question Bot app. This time I build a data source object to keep track of a conversation between a user and a chatbot.

8. Element Quiz

An app for studying chemical elements with flashcards and a quiz.This app is using the single-path UI update pattern to manage the UI state.I created the interface, added data, handle user interaction, getting keyboard input, and displaying the results.
