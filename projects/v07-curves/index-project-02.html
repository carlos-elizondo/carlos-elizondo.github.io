Crafting Curves in the Digital Realm: A Three.js-Powered Visualization of Bezier, B-Spline, and Hermite Curves

In this project I implemented algorithms to compute Bezier, B-Spline, and Hermite curves. Using Three.js to display them in a web browser.

Description

I am using Three.js, a cross-browser Javascript library to display graphics using WebGL.

The following curves are supported: Bezier, B-Spline, and Hermite.

There are two canvases the left one is for curve drawing and the right one is for surface rendering. You can toggle between wireframe and solid rendering modes.

You can also save the point controls or the mesh as an OBJ file using the control panel at the top-right corner.

The nSegment control defines how many line segments should be used to approximate the curve.

You can drag the red rectangles on the left canvas to interactively change the curve and add points using CTRL+Click on the canvas.

Instructions

  • In order to activate the key controls you will have to click on the canvas first.
  • Please look for the key/mouse controls after the canvases.
