Key/Mouse Controls
  • 'z': draw Bezier
  • 'b': draw BSpline
  • 'm': draw Hermite
  • '=': increment nSegment
  • '-': decrement nSegment
  • 'e': erase all control points
  • 'l': show control lines
  • 'c': toggle close/open curve (applicable to BSpline and Hermite only)
  • 'a': toggle show/hide rotation axis
  • 's': generate or update surface (or press space bar)
  • 'w': toggle wireframe/solid rendering mode
  • 'f': toggle flat/smooth rendering mode
  • CTRL+Click : add a control point (or ALT+Click, COMMAND+Click)
  • SHIFT+Click: delete an existing control point
  • Arrow keys : move rotation axis left/right
  • Number keys: load preset control points (1 through 5)
  • Auto Mesh (on control panel): keep mesh updated whenever curve changes
